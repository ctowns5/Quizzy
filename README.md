# Quizzy

### a short coding quiz game
When you start the quiz the timer starts counting down from 75 seconds and a question is presented to be answered. If you answer all questions correctly in less than 75 seconds you win and your score and initials will be displayed on the high scores page. If you answer a question incorrectly 10 seconds will be taken off the timer.
